# attendance-compilation-system
Smart attendance management system built with React and Django for automatic handwritten attendance extraction, compilation, analytics, and report generation.
Smart Attendance ERP is an AI-powered attendance management system for colleges and universities. It converts scanned handwritten attendance sheets into digital records using OCR technology. Faculty can upload PDF or image-based attendance sheets, and the system automatically detects Present, Absent, and Leave marks. It compiles attendance from multiple faculty using majority logic and generates monthly attendance registers, attendance percentages, subject-wise reports, and internal marks. Features include Excel/CSV/PDF export, analytics dashboard, OCR correction, and role-based access for Admin, Faculty, and HOD users. Built with React.js, Tailwind CSS, Django, SQLite, Tesseract OCR, and OpenCV.
